# angularClients

angularClients is a simple CRM write in angularjs with express.
It was did for a school project.

## Requirements

* `npm install`
* `npm install nodemon`
* `npm install mongodb --global`

## Run
### Server (/)
`nodemon js/server.js`
### Web client (/)
`python -m SimpleHTTPServer`
